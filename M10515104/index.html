產品或服務簡介

    本組提出之服務宗旨為「運用政府 Open Data，並整合職場經驗談；打造公開透明的職場資訊，以提升台灣就業品質。」

 

    本服務保有類似平台之功能下，首創「整合」政府Open Data正、負面資訊，以供求職者參考之。正面資訊，如：優良企業、績優企業，使得求職者得知優良企業，此舉可減少該企業之行銷開銷；負面資訊，如：違反勞基法、性別工作平等法等法律，讓非法公司無所遁形。

 

    未來更可搭配資訊檢索技術(IR)分析使用者回饋，過濾擾亂平台秩序發言者，如：蓄意攻擊等；且有鑑於資安就是國安，故企業更應重視資安問題，未來更可結合漏洞平台資訊，來檢視企業是否確實重視之。

 

    服務將提供有實習需求之學生、社會新鮮人、求職者等，瞭解企業的另一面，此服務將提供選擇公司時的重要參考依據。

產品或服務說明

 

 

壹、    緣起與創作目的

    每年皆有大量畢業生投入職場行列，而因學生未有職場經驗，時常透過網際網路取得資訊，然而網路資訊可信度低，如何提供最公正之資訊，是我們提供服務的起源。

 

    新聞媒體為民眾接收資訊的重要來源之一，但新聞內容保留在民眾的記憶中卻只有短暫幾週，為此我們提出「整合」新聞中涵蓋企業之正負面資訊，以凡走過必留下痕跡的概念下，供每位求職者參考企業之歷史。

 

    為以客觀的角度出發，避免以訛傳訛之謠言，且不傷企業之信譽之考量下，以政府公告違反勞基法等法律之Open Data為原則，提供最公正的職缺交流分享平台，避免道聽塗說，耽誤未來之職涯發展。

  

貳、    市場調查與產品或服務定位(需含1~2同質性產品或服務比較說明)

類似平台：求職天眼通、各行各業薪資查詢網

   

    上述該二平台，皆依據使用者回饋之資訊，而未有第三方公正之企業違法之資訊，使用者即不可進行最透明化之參考。請參考下表，本平台服務與該二已存平台以公正性、全命性進行比較。

肆、    產品或服務特色說明

    本平台「服務特色」，主以「整合」政府Open Data之資訊，供求職者參考，完全以第三方的角度著手。

   

    特色一：本服務將提供企業「正、負」面資訊，讓學生實習、求職者有效率地取得企業歷史資訊。

　　

    特色二：雖本服務主打供求職者參考，但亦可供投資者、經營者參考之。投資者：可觀察企業之形象或經營態度，來決定該公司是否值得投資。經營者：藉由觀察其他經營者之經營理念，是否有值得學習的企業，亦可加以拜訪之，或以他企業做為借鏡。

   

    特色三：視覺化效果，讓數值深入內心。以歷年統計圖表之方式提供使用者最公正的參考指標，得知該企業是否為優良企業，或依然持續違反相關法律。

   

    特色四：平台服務完全採不收費機制，供求職者享受平台資源。因平台經營仍需經費，故經費來源以 B2B 廣告收入，針對企業申請廣告進行嚴格把關，如以出現於本平台劣質資料庫中之企業，皆不可申請，確保在此平台的廣告皆為優良企業。

 

    除上述之特色外，平台服務可於校園徵才(如：台大、台科等)或大型徵才活動(如近期之臺北市就業服務處於南港軟體園區中型徵才活動)中，可進一步推廣之，使得多數人可得知此平台，以「成為求職者資訊參考、相互交流之首選平台」為目標，為臺灣之勞資雙方共創雙贏之平台服務。

 

伍、產品或服務功能(請條列之，此項為評選時測試產品或服務功能之依據)

平台服務功能如下：

1.      結合 Open Data 之 「勞動基準法洽詢管道」 與「Google Map 」提供視覺化地理位置。

2.      提供Open Data 最新 之 「勞動基準法適用公告」供使用者參考。

3.      提供 Open Data 最新 之 「各縣市政府違反勞動基準法事業名單」、「違反性別工作平等法名單」，並依據歷年紀錄提供統計圖表。

4.      提供 Open Data 最新 之 「優良企業」、「績優企業」、「優質企業認證合格業者名冊」、「一般優質企業名冊」，並依據歷年紀錄提供統計圖表。

5.      提供使用者薪資、面試、環境等訊息交流之平台。

6.      整合關於企業之新聞資訊，提供歷年文章參考。

 

 

陸、未來規劃(如產品或服務功能擴充等)

為提供更優質的服務之未來規劃如下：

1.          搭配資訊檢索技術(IR)，分析使用者回饋，亦過濾擾亂平台秩序言論。

2.          整合其他相關平台資源，如：PTT等，成為求職之第一首選平台。

3.          整合資安漏洞平台資源，如：HITCON ZeroDay，以因應資安即國安之趨勢。資安做不好，恐會動搖企業之生存空間，亦會造成需與企業共體時艱之情況。
